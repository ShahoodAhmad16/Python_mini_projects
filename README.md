# Python Mini Projects

This repository contains a collection of small Python projects for learning and practice purposes. Each project focuses on basic Python programming concepts and provides hands-on experience with data handling, file operations, and object-oriented programming.

## Projects Included

1. **Contact Book Manager**  
   A simple program to manage contacts. You can add, view, update, and delete contact information.

2. **Library Management System**  
   A system to manage library operations such as adding books, issuing books, returning books, and viewing the library database.

3. **Student Report Card Program**  
   A program to manage student grades and generate report cards based on their marks.

4. **Task To-Do List Manager**  
   A program to manage tasks, allowing you to add, view, update, and delete tasks from your to-do list.

## Features

- Easy-to-understand Python code
- Practice with basic data structures like lists and dictionaries
- File handling in Python
- Basic object-oriented programming concepts
